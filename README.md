# chatbot-fifa-extension
Extension for zoozl chatbot on FIFA World Cup betting game

## Usage
This module is an extension module for zoozl chatbot

The plugin can be run using zoozl:

```
python3 -m zoozl --conf config.toml
```

A minimum `config.toml` should include `chatbot_fifa_extension` as an extension:

```toml
[extensions]
names = ["chatbot_fifa_extension"]
```
